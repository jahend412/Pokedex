# Pokédex App

## About
This app is a pokedex that loades data from an external Pokémon API to allow users to search through the original 151 Pokémon to find details about their favorite Pokémon.
---------------------

##  Languages and Tech used
 -HTML
 -CSS
 -Javascript
 -Bootstrap
 -Fetch polyfill
 -Promise polyfill
